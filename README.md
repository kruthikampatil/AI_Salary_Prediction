# AI_Salary_Prediction Project

This project predicts employee salary using Machine Learning.

## Tools Used
- Python
- Pandas
- Matplotlib
- Scikit-learn
- Jupyter Notebook

## Project Workflow
1. Loaded dataset using Pandas
2. Performed data analysis
3. Created visualizations
4. Applied Machine Learning model
5. Predicted employee salary
6. Compared actual vs predicted salary

## Dataset Features
- Age
- YearsExperience
- Education
- Salary

## Machine Learning Model
- Decision Tree Regressor

## Visualizations
- Salary Distribution Histogram
- Education Distribution Chart
- Actual vs Predicted Salary Graph

## Output
- Predicted employee salary based on input features
- Model evaluation using Mean Absolute Error
